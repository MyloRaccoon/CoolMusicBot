# Cool Music Bot
A very cool Discord Music Bot because youtube takes down every one of them so I'm making it myself.

## Commands
### Play <querry_song>
Play a song from youtube or adds it to queue

### Queue
Get the current playing song and songs in queue

### Skip
Skip the current song
